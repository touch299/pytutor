# PyTutor — Python 执行轨迹可视化工具
在网页上编写 Python 代码，逐行查看变量变化与执行流程，适合 Python 初学者理解代码执行逻辑。

## 功能特性
- 左侧代码编辑，右侧实时展示完整执行时间线
- 自动追踪变量赋值、循环迭代、条件分支、print 输出
- 点击执行步骤，对应代码行同步高亮
- 内置常见报错的中文解释，辅助定位代码问题

## 环境要求
- Python 3.7 及以上版本
